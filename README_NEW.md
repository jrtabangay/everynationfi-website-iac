# Every Nation Finland Website

A beautiful, modern church website built with **pure HTML, CSS, and JavaScript** — perfect for GitHub Pages hosting!

## 🎯 What's Inside

✨ **index.html** - Complete website (all sections in one file)  
🎨 **styles.css** - All styling (responsive, mobile-friendly)  
⚡ **script.js** - Interactive features (mobile menu, contact form)  

## 🚀 Quick Start

### 1. Local Preview
Open `index.html` directly in your browser, or use a local server:
```bash
# Python
python -m http.server 8000

# Node.js
npx http-server
```

### 2. Deploy to GitHub Pages
See **[GITHUB_PAGES_GUIDE.md](GITHUB_PAGES_GUIDE.md)** for step-by-step instructions

### 3. Customize
Edit `index.html` to update:
- Church name and tagline
- Cities and locations
- Events and announcements
- Contact information

## ✨ Features

- 🎨 Responsive design (mobile, tablet, desktop)
- ⚡ Fast loading (no build process needed)
- 📱 Mobile menu navigation

## 🎨 Customize Colors

Edit `styles.css`:
```css
:root {
    --primary: #1a1a1a;      /* Change main color */
    --secondary: #f97316;    /* Change accent color */
    --accent: #ffffff;       /* Change white color */
}
```

This project requires **NOTHING**:
- ✅ No Node.js
- ✅ No npm packages

- ✅ Pure HTML/CSS/JavaScript

Works in any modern browser!

## 🔄 Update Your Site

1. Edit `index.html`, `styles.css`, or `script.js`
2. Save changes
3. Commit and push to GitHub:
   ```bash
   git add .
   git commit -m "Update website"
   git push origin main
   ```
4. Changes live in 1-2 minutes!

## 🚀 Go Live

**See [GITHUB_PAGES_GUIDE.md](GITHUB_PAGES_GUIDE.md) for complete setup instructions**

In short:
1. Push repo to GitHub
2. Enable GitHub Pages (Settings → Pages)
3. Select "main" branch and "/" folder
4. Done! 🎉
